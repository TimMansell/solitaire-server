# Tests

This folder contains end-to-end tests and fixtures. Unit tests are located inside `__tests__` in their respective folders.

## Unit tests

1. [App](/src)
1. [Components](/components)
1. [Service](/service/solitaire)

## Running tests

### Run unit tests

```
yarn test:unit
```

### Run end-to-end tests

```
yarn test:e2e
```
